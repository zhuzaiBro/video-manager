# Video Platform PRD v1.0

## Project Overview

构建一个支持付费视频播放的视频平台。

系统支持：

* 视频上传
* 自动转码 HLS
* COS 存储
* CDN 分发
* 播放授权
* 播放时长限制
* 防盗链
* 后续支持动态水印

技术目标：

* 支持 20GB 单视频上传
* 支持 100000+ 切片访问
* 支持 CDN 分发
* 支持水平扩容
* 支持后续会员体系接入

# Database Schema

## videos

```sql
create table videos (
    id bigint generated always as identity primary key,

    title text not null,

    description text,

    source_file text not null,

    cos_prefix text,

    m3u8_path text,

    duration integer,

    width integer,

    height integer,

    fps numeric(10,2),

    segment_count integer,

    status text not null default 'waiting',

    error_message text,

    created_at timestamptz default now(),

    updated_at timestamptz default now()
);
```

---

## user_video_usage

```sql
create table user_video_usage (
    id bigint generated always as identity primary key,

    user_id bigint not null,

    usage_date date not null,

    watch_seconds integer not null default 0,

    created_at timestamptz default now(),

    updated_at timestamptz default now(),

    unique(user_id, usage_date)
);
```

# Video Processing Flow

## Step 1

FFprobe

Command:

```bash
ffprobe \
-v quiet \
-print_format json \
-show_format \
-show_streams \
input.mp4
```

Store:

* duration
* width
* height
* fps

---

## Step 2

Generate HLS

Use fMP4.

Command:

```bash
ffmpeg \
-i input.mp4 \
-c copy \
-hls_segment_type fmp4 \
-hls_time 6 \
-hls_playlist_type vod \
-hls_flags independent_segments \
output/index.m3u8
```

Expected Output

```text
index.m3u8

init.mp4

seg_000001.m4s

seg_000002.m4s
```

---

## Requirements

Must not:

* re-encode video
* reduce quality
* drop frames

Must:

* keep original quality
* keep original fps

# Watch Time Limitation

Daily Limit

```text
8 Hours
```

Equivalent

```text
28800 Seconds
```

---

Segment Duration

```text
6 Seconds
```

---

Each successful segment access

```text
+6 Seconds
```

---

If

```text
watch_seconds >= 28800
```

Return

```http
403 Forbidden
```

Response

```json
{
  "code":"WATCH_LIMIT_EXCEEDED",
  "message":"Daily watch limit exceeded"
}
```
